Projet DevOps
Description

Projet DevOps complet mettant en place un workflow CI/CD (lint, tests Jest, build, push, déploiement) pour une application de gestion de listes/tâches, conteneurisée avec Docker et déployée automatiquement sur Kubernetes (k3d/Kind). La persistance est assurée par MariaDB avec volume, et les secrets (credentials) sont gérés via des variables GitLab CI injectées dans les pods, avec possibilité d’ajouter monitoring (Prometheus/Grafana) et tests E2E (Cypress).
