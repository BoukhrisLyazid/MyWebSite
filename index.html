Puissance 4
Description

Implémentation numérique du jeu Puissance 4 avec une IA basée sur MiniMax et l’élagage alpha-bêta. Le jeu permet de jouer en solo contre l’IA ou à deux joueurs, avec une interface interactive.

Fonctionnalités
  • Mode solo contre IA (MiniMax + alpha-bêta)
  • Évaluation heuristique du plateau pour optimiser les décisions
  • Détection automatique des victoires (horizontal/vertical/diagonales)
  • Interface utilisateur interactive
